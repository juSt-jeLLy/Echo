# Requirements: UI Typography Scale

## Introduction
Increase the visual weight and size of key UI elements across CityPanel and EraCard so the experience feels more cinematic and immersive.

## Requirements

### Requirement 1: Bigger cards
- CityPanel and EraCard max-width increases from `max-w-3xl` to `max-w-4xl`

### Requirement 2: Bigger era text in EraCard
- Era label (e.g. "1850") increases from `text-xl` to `text-3xl`
- Era tagline increases from `text-[11px]` to `text-sm`
- Era card padding increases from `p-3.5` to `p-5`

### Requirement 3: Bigger event name in CityPanel
- Event name badge text increases from `text-sm font-semibold` to `text-xl font-bold`
- Radio icon increases from `size={11}` to `size={14}`
- Atmosphere text increases from `text-sm` to `text-base`

### Requirement 4: Bigger era label in CityPanel header
- Era label in header (`{era.label} · {city.country}`) increases from `text-xs` to `text-sm`
